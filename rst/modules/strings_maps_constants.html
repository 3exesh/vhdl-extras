strings_maps_constants¶

extras/strings_maps_constants.vhdl

Dependencies¶

strings_maps, characters_latin_1

Description¶

This package provides constants for various character sets from the range of Latin-1 and mappings for upper case, lower case, and basic (unaccented) characters. It is a clone of the Ada‘95 package Ada.Strings.Maps.Constants.

Constants¶
